# 更新日志

## v0.0.2 (2026-04-03)

### 新增
- **模型测速功能** — 对比多个 GLM 模型的响应速度（首 Token 时间、总耗时、生成速度）
- **并发测试** — 支持 1~8 线程并发测速
- **模型管理** — 预设 glm-4.7/glm-5/glm-5-turbo/glm-5.1，支持手动添加/删除模型，配置跨会话持久化
- **Key 脱敏显示** — API Key 默认中间遮掩（前6后4），点击切换明文/密文
- **后台测速** — 关闭面板后测速继续执行，完成后弹 VS Code 通知，点击可查看结果
- **进度恢复** — 重新打开面板可恢复显示测速进度
- **状态栏监控** — Token 使用百分比进度条 + 重置倒计时
- **详细使用报告** — 配额限制、模型使用量、工具使用量
- **多时间维度** — 24 小时/7 天/30 天查询范围
- **自动刷新** — 1~60 分钟可配置刷新间隔
- **窗口聚焦刷新** — 切回 VS Code 窗口自动刷新

### 技术细节
- 测速使用智谱 AI Anthropic 兼容接口，不消耗账户余额
- 模型配置通过 VS Code globalState 持久化存储
- SSE 流式解析，实时计算首 Token 时间和生成速度

## v0.0.1 (2026-04-02)

### 新增
- GLM Key Monitor 初始版本
- 状态栏实时显示 Token 使用百分比和重置倒计时

- 自动刷新和窗口聚焦刷新
- 颜色分级显示（绿/黄/红）
